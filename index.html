Our Features

Parallax Scrolling

Adds a sense of depth by fixing the background while the page scrolls.

Interactive Cards

Hover over cards to see a smooth scaling and shadow effect.

Animated Buttons

Buttons glow and grow when you hover over them to show they're interactive.

How These Features Were Built

The parallax effect is done using background-attachment: fixed in CSS, so the background stays in place while the content scrolls.

We use Bootstrap cards and add hover effects in CSS, like scaling the card and adding a shadow.

Buttons are styled with CSS transitions to change size, color, and glow when hovered over.
